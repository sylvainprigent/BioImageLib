Detailed Description

class BLCORE_EXPORT that allows to read/write param into a file To use it for reading: call the constructor 'ParamFile(fName)', then 'load()' to load parameters, and 'getValueOfKey' to read any parameter individually.

To use it for writing: call the constructor 'ParamFile(fName)', then call 'load()' only if the file exist, then set param with 'setValueOfKey' and finally call 'updateFile()' to save modification into the file.
