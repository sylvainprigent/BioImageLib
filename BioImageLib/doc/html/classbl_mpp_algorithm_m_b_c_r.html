Detailed Description

Implement the multiple birth and cut algorithm with binary (ie infinity) repulsion as possible interaction This implementation is based on the publication "A 3D segmentation algorithm for ellipsoidal shapes. Application to nuclei extraction." Soubi{`e}s, Emmanuel and Weiss, Pierre and Descombes, Xavier ICPRAM - International Conference on Pattern Recognition Applications and Methods, Barcelona, 2013 The graph used in this implementation is a simplification of the graph decribed in the publication above.
