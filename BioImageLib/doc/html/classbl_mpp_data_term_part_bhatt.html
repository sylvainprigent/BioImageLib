Detailed Description

Class that define an Marked Point Process data term as the Bhattacharyya distance calculated between the distributions of the pixels inside and the pixels in the border of a shape The shape bordure is partitioned into 4 connected regions, and at least 3 of these regions must have a Bhattacharyya distance greater than the threshold for the data term to be negative This data term has been made for the axon data set of Caroline.
