Detailed Description

Class defing the basic functions needed to implement a Marked Point Process Algorithm. This class need a Shape generator derivated from blMppGeneratorShape to generate shapes, blMppdataTerm to compute the dataTerm term, and blMppInteraction to evaluate interactions between pairs of objects.
